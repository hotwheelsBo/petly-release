# Petly

Petly is a desktop companion app that keeps a small pixel pet on your screen while you work.

## Download

Latest desktop builds:

- macOS Apple Silicon: https://petly-static.hpaiapp.com/download/Petly-latest-mac-arm64.dmg
- macOS Intel: https://petly-static.hpaiapp.com/download/Petly-latest-mac-x64.dmg
- Windows x64: https://petly-static.hpaiapp.com/download/Petly-latest-win-x64-setup.exe
- Linux x64: https://petly-static.hpaiapp.com/download/Petly-latest-linux-x86_64.AppImage
- SHA256 checksums: https://petly-static.hpaiapp.com/download/SHA256SUMS

GitHub Releases may also provide the same installers for users who prefer GitHub downloads.

## Features

- Desktop pet companion with screen mode and Dynamic Island mode on supported MacBook Pro devices.
- Pet catalog, install/remove/default pet management, and localized pet information.
- Growth system with level, hunger, food, check-in, interaction, and pet sounds.
- Cloud account sync and recovery key support.
- VIP membership unlocks all pets and AI chat.
