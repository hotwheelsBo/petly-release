## What Petly Does

Petly is designed to be present without getting in the way:

- It stays on your desktop as a small companion.
- It reacts to interaction and daily state.
- It can switch into a compact Dynamic Island mode on supported MacBook Pro devices.
- It lets you install and manage different pets.
- It grows through feeding, check-ins, interaction, and chat.
- It syncs pet data through a cloud account so recovery is possible.

## Core Features

### Desktop Companion

Keep a small pet on screen while you work. Petly is tray-first and designed for repeated daily use rather than a large main window.

### Pet Collection

Browse the catalog, install pets, set a default pet, and keep your collection synced with your cloud account.

### Growth Loop

Feed your pet, check in, interact with it, and level it up over time. Petly is built around small, low-friction rituals.

### AI Chat

VIP users can chat with their pet through Petly cloud AI chat. Each pet can carry personality information so the conversation feels more like a companion than a generic assistant.

### Cloud Account and Recovery

Petly automatically creates a cloud account and provides a user number and recovery secret. Keep the recovery secret private; it can be used to restore account data.

## Install Notes

Current desktop artifacts are unsigned and not notarized. macOS and Windows may show security warnings on first launch.

Petly uses cloud services for:

- Account sync and recovery.
- Pet data and growth state.
- Membership entitlement.
- Payment order status.
- AI chat.
