# Changelog

## v1.0.0

Initial public desktop release.

Included:

- Desktop companion pet window.
- Control Center with Dashboard, Pets, Settings, and Growth tabs.
- Cloud account auto-login, sync, and recovery.
- Pet catalog with localized Chinese/English metadata.
- Shared pet voice cues for interactions.
- VIP membership paywall and WeChat Native order generation.
- VIP-gated AI chat through Petly cloud services.

Known limitations:

- Desktop artifacts are currently unsigned and not notarized, so macOS and Windows may show security warnings.
- Paid-flow release should be considered complete only after one real WeChat payment callback is verified in production.
