# Petly v1.0.0

Petly v1.0.0 desktop release.

## What's Included

- Desktop companion app with cloud account sync.
- Control Center pet catalog, growth, settings, and membership paywall.
- VIP-gated AI chat through the cloud message service.
- WeChat Native membership order generation and QR-code display.

## Download

- macOS Apple Silicon: `Petly-latest-mac-arm64.dmg`
- macOS Intel: `Petly-latest-mac-x64.dmg`
- Windows x64: `Petly-latest-win-x64-setup.exe`
- Linux x64: `Petly-latest-linux-x86_64.AppImage`
- Checksums: `SHA256SUMS`

## Known Limitations

- Desktop artifacts are currently unsigned, so OS security warnings may appear.
- Paid-flow release requires one real WeChat payment verification for callback delivery and VIP entitlement refresh.
